How the World Wide Web works

The web is a series of networks that link your computer to other computers and servers. Using a web browser from your personal computer, you can type in url link to a webpage and a signal (HTTP Request) is sent through your internet provider and routed to the server that is hosting that webpage. The server retrieves the appropriate HTML document and returns it to your computer, where the browswer reads it.

HTML or Hypertext Markup Language

Most content on the internet is made up of HTML. It contains tags, elements, and attributes. HTML is the language of your browser. It contains text and markup (how it looks).

Tags, Elements and Attributes

Tags are names surrounded by "< >" which define the content inside and tell it what to do. Attributes are values inside the opening tag that provide additional instructions for the content. Some examples of tags and attributes are b for boldb>, em for italics, a href = for links, and br for
breaks
Elements are groupings of HTML code. Individual tags can be an element, or the entire body can be one.

Inline vs Block

HTML can be displayed in either

block
or inline elements. Inline only takes up as much width as necessary whereas block elements take up the entire width allowed. Block elements can be described as having an invisible box around them that defines the area.
